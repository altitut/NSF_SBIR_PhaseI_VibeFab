\subsection*{Budget Consistency Review}

Before finalizing the budget, confirm:

\begin{itemize}
    \item[$\square$] Every budgeted person appears in the Project Description
    or supporting personnel materials.
    \item[$\square$] Person-months match the Company/Team section.
    \item[$\square$] Personnel roles match the R\&D tasks.
    \item[$\square$] Consultants match their commitment letters.
    \item[$\square$] Materials and equipment match the experimental plan.
    \item[$\square$] Cloud and model costs match the planned benchmarks.
    \item[$\square$] Subaward amounts match the subaward budget and scope.
    \item[$\square$] Indirect costs use the correct rate and base.
    \item[$\square$] Fee uses the correct calculation.
    \item[$\square$] Total amount matches the final Research.gov budget.
\end{itemize}
